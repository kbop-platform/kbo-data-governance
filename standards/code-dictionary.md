# 코드 사전

> 최종수정: 2026-02-25 | 출처: analysis/code-analysis.md, raw/column-metadata.json

## 1. 개요

KBO 데이터 시스템에서 사용하는 모든 코드값을 정의한다.
코드값은 **영문 대문자**를 원칙으로 하며, 레거시 시스템의 실제 데이터 분포(건수)를 함께 기재한다.

> **"추정" 표기 안내**: "(추정)"으로 표기된 코드값은 데이터 분포 패턴에서 유추한 것이며, S2i 확인 전까지 참고용이다. 미확인 코드 전체 목록은 [부록 C](#c) 참조.

→ 참고: [명명 규칙](./naming-rules.md) — 코드 컬럼 네이밍
→ 참고: [약어 사전](./abbreviations.md) — 코드에 사용된 약어 의미
→ 참고: [ID 체계](./id-system.md) — 식별자와 코드의 구분

## 3. 포지션/라인업 코드

### 3.1 position_cd — 포지션 (XY 형식)

사용 테이블: ENTRY, DEFEN

**형식**: 2자리 — `X` (교체순번) + `Y` (포지션)

```
XY
│└─ 포지션 (1~9, D, H, R)
└── 교체순번 (1=선발, 2=1차 교체, 3=2차 교체, ...)
```

#### 포지션 번호

| Y | 포지션 | 영문 |
|---|--------|------|
| `1` | 투수 | Pitcher (P) |

| `3` | 1루수 | First Base (1B) |
| `4` | 2루수 | Second Base (2B) |
| `5` | 3루수 | Third Base (3B) |
| `6` | 유격수 | Shortstop (SS) |
| `7` | 좌익수 | Left Field (LF) |
| `8` | 중견수 | Center Field (CF) |
| `9` | 우익수 | Right Field (RF) |
| `D` | 지명타자 | Designated Hitter (DH) |
| `H` | 대타 | Pinch Hitter (PH) |
| `R` | 대주자 | Pinch Runner (PR) |
